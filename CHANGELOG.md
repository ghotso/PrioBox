# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-11-10
### Added
- Initial project setup for PrioBox with multi-account support, secure credential storage, and MVP inbox/compose/settings flows.
- Message detail view with VIP toggles, connection testing for IMAP and SMTP, and onboarding state for first-time users.
- Localization scaffolding (English and German) and onboarding empty-state flow.
